Aplikasi Pembaca Metadata Foto

Ekstrak informasi metadata dari foto dengan mudah dan cepat. Validasi data dokumentasi lapangan agar akurat, mudah diverifikasi, dan siap olah sesuai kebutuhan.

Mulai Panduan

Tentang Aplikasi

Aplikasi ini dirancang khusus untuk membaca metadata dari foto, terutama informasi penting yang diperlukan untuk validasi data dokumentasi lapangan.

Dengan aplikasi ini, Anda dapat dengan mudah mengekstrak informasi seperti nama file, tanggal & waktu pengambilan gambar, koordinat GPS, dan deskripsi foto sesuai label EXIF.

Aplikasi ini membantu proses validasi data dokumentasi lapangan agar akurat, mudah diverifikasi, dan dapat langsung diolah sesuai kebutuhan.

Informasi yang Diekstrak
  • Nama file
  • Tanggal & Waktu pengambilan gambar
  • Koordinat GPS (Latitude & Longitude)
  • Format koordinat gabungan
  • Deskripsi sesuai label EXIF pada foto

Ketentuan Penggunaan

Aplikasi hanya dapat bekerja pada:

  • File foto asli (original) langsung dari kamera/HP yang diambil menggunakan aplikasi Timestamp camera
  • Foto yang belum mengalami proses penghapusan metadata
Sumber File yang Didukung:
  • External drive (Flashdisk / Harddisk)
  • File yang ditransfer menggunakan Kabel USB
  • File yang di Share di Google Drive
  • Local storage

Catatan: Jika metadata hilang, maka data waktu dan koordinat tidak dapat ditampilkan.

Batasan Penggunaan

Aplikasi tidak dapat membaca metadata dari:

  • Foto dikirim melalui WhatsApp / Telegram
  • Foto hasil Screenshot
  • Foto yang sudah di-compress oleh aplikasi tertentu
  • Foto yang metadata-nya telah dihapus atau diubah
  • File yang bukan gambar (video, dokumen, dll)
Penting: Pastikan foto yang akan diproses adalah foto asli yang masih mengandung metadata EXIF.

Fitur Utama Aplikasi

Input Folder

Memilih folder berisi kumpulan foto untuk diproses otomatis. Aplikasi akan membaca semua foto yang valid dalam folder tersebut.

Start Processing

Memulai proses ekstraksi metadata setelah folder dipilih. Aplikasi akan mengekstrak informasi dari semua foto yang valid.

Copy All Data

Menyalin semua data pada tabel ke clipboard untuk digunakan di aplikasi lain seperti Excel atau Google Sheets.

Preview Tabel Metadata

Menampilkan metadata yang diekstrak dalam format tabel yang rapi dan mudah dibaca.

Export KML

Menghasilkan file KML untuk ditampilkan di Google Earth atau aplikasi GIS lainnya.

Export Excel

Menyimpan data metadata menjadi file Excel untuk analisis dan pelaporan lebih lanjut.

Panduan Penggunaan

1
Pilih Folder Berisi Foto

Klik tombol "Input Folder" dan pilih folder yang berisi foto-foto yang ingin diproses. Pastikan foto dalam folder adalah foto asli dengan metadata EXIF yang masih utuh.

2
Mulai Proses Ekstraksi

Setelah folder dipilih, klik tombol "Start" untuk memulai proses ekstraksi metadata. Aplikasi akan membaca semua foto yang valid dalam folder tersebut.

3
Lihat Hasil di Tabel

Setelah proses selesai, metadata akan ditampilkan dalam tabel. Anda dapat mengklik header tabel untuk mengurutkan data berdasarkan kolom tertentu.

4
Ekspor Data

Gunakan tombol "Export KML" untuk membuat file KML yang dapat dibuka di Google Earth, atau "Export Excel" untuk menyimpan data dalam format spreadsheet.

5
Reset atau Selesaikan

Setelah selesai, Anda dapat menggunakan tombol "Reset" untuk menghapus semua data dan kembali ke kondisi awal, atau "Exit" untuk menutup aplikasi.
